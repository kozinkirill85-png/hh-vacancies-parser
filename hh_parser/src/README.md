# hh.ru Вакансии Парсер

Проект для получения данных о компаниях и вакансиях с hh.ru и сохранения их в PostgreSQL.

## Установка

1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Создать `config.py` с параметрами подключения к БД (см. пример в описании)
4. Запустить: `python main.py`

## Функционал
- Получение данных через API hh.ru
- Сохранение в PostgreSQL
- Анализ вакансий: средняя ЗП, фильтрация по ключевым словам и т.д.
